# Anonimo Menu Web

Sitio web responsive para el menu de Anonimo Restaurante Cafe.

## Contenido

- Landing principal con identidad visual de la marca
- Seccion de menu por categorias
- Filtros por tipo de producto
- Accesos directos a WhatsApp e Instagram

## Archivos principales

- `index.html`
- `styles.css`
- `script.js`

## Ejecutar en VS Code

Abre la vista **Run and Debug**, selecciona **Abrir menú Anónimo** y pulsa el botón verde o `F5`.
La configuración incluida abre `index.html` en Microsoft Edge.
